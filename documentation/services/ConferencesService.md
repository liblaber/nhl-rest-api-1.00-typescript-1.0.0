## getConferences

This only retrieves active conferences. For inactive conferences, use `/conferences/{id}`.

- HTTP Method: `GET`
- Endpoint: `/conferences`

**Return Type**

`Conferences`

**Example Usage Code Snippet**

```typescript
import { Nhl } from 'nhl';

(async () => {
  const nhl = new Nhl({});

  const { data } = await nhl.conferences.getConferences();

  console.log(data);
})();
```
