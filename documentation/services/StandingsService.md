## getCurrentStandings

Retrieve the standings as of the current moment.

- HTTP Method: `GET`
- Endpoint: `/v1/standings/now`

**Return Type**

`any`

**Example Usage Code Snippet**

```typescript

(async () => {
  const nhl = new Nhl({});

  const { data } = await nhl.standings.getCurrentStandings();

})();
```
