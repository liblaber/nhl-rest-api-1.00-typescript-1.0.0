## getStandings

- HTTP Method: `GET`
- Endpoint: `/standings`

**Parameters**

| Name   | Type   | Required | Description                       |
| :----- | :----- | :------- | :-------------------------------- |
| season | string | ❌       | Standings for a specified season. |
| date   | string | ❌       | Standings on a specified date.    |

**Return Type**

`Standings`

**Example Usage Code Snippet**

```typescript
import { Nhl } from 'nhl';

(async () => {
  const nhl = new Nhl({});

  const { data } = await nhl.standings.getStandings({

  });

  console.log(data);
})();
```
