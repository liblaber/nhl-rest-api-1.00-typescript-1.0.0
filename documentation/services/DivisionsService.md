## getDivisions

This only retrieves active divisions. For inactive divisions, use `/divisions/{id}`.

- HTTP Method: `GET`
- Endpoint: `/divisions`

**Return Type**

`Divisions`

**Example Usage Code Snippet**

```typescript
import { Nhl } from 'nhl';

(async () => {
  const nhl = new Nhl({});

  const { data } = await nhl.divisions.getDivisions();

  console.log(data);
})();
```
